# Slimy Tiles
### Created by James Wisniewski
The final project created for IT 265.

Github Repository - https://github.com/jimsgims/IT265-GameProject

Sound effects from freesound.org.

## CONTROLS

W - move north

A - move west

S - move south

D - move east

Mouse1 - selection tool
